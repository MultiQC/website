MultiQC Workshop Phil Ewels, 21st October 2016

Welcome to the SBW 2016 MultiQC Workshop!

There are two sets of excercises for this workshop. There will be time to do the first set during the workshop, we probably won't have time for the second set - these are optional for you to do in your spare time if you wish (or if you finish the first set quickly).

Part One

  • Basic MultiQC usage.
  • Running MultiQC on different files, basic command line flags.
  • Using reports, exporting plots and using parsed results files.

Part Two (optional)

  • Advanced MultiQC usage.
  • Customising default behaviour, file search patterns.
  • Sample name cleaning, fixing overwritten sample names.
  • Extra report metadata, customising report display.
