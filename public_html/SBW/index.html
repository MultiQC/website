MultiQC Workshop

Welcome to the SBW 2016 MultiQC Workshop!

See if you can work through the examples below - there's no rush, just do as many as you can in the available time.

1. Install MultiQC

The first task is to get up and running with MultiQC. There are a few ways to do this:

  • If you want to use MultiQC on UPPMAX
    • MultiQC is already installed as an environment module: module load bioinfo-tools module load MultiQC
  • Install yourself from the Python Package Index:
    • pip install multiqc
  • If you have conda installed, from Bioconda:
    • conda install -c bioconda multiqc

Having difficulties? Take a look at the MultiQC documentation for more detailed instructions.

2. Check that MultiQC is installed properly

You can test MultiQC is working properly by printing the version: multiqc --version or help text: multiqc -h

3. Try on a basic dataset

Ok, you're ready to run this for real. Grab an example dataset, unzip it and try running MultiQC.

Get some help

wget http://multiqc.info/examples/rna-seq/data.zip
unzip data.zip
cd data
multiqc .

Running with loads of samples

MultiQC can handle really huge numbers of samples. This is great if you're working with single-cell data, or other large datasets. The conquer single-cell datasets all come with MultiQC reports. Try running MultiQC on the logs from their largest dataset, GSE57872 (Patel et. al.) with 864 cells. Download the logs files here.
